# IT Help Desk Homelab Portfolio

This repository documents real-world IT support scenarios I built and resolved in a Windows 11 virtual lab environment while preparing for entry-level Help Desk / Desktop Support roles and CompTIA A+ certification.

The goal of this project is to demonstrate practical troubleshooting skills, structured documentation, and familiarity with common IT support tools and workflows.

---

## 🧰 Skills Demonstrated

- Windows 10 / 11 Troubleshooting
- Network Troubleshooting (DHCP, IP configuration, connectivity)
- Printer & Print Spooler Management
- User Support & Incident Handling
- Windows Services (services.msc)
- Device Manager & Driver Troubleshooting
- Basic Active Directory concepts (lab simulated)
- Root Cause Analysis
- Technical Documentation

---

## 📁 Repository Structure

- **Tickets/** → Documented help desk incidents (step-by-step troubleshooting)
- **Screenshots/** → Supporting visuals for troubleshooting steps
- **Documentation/** → Notes on tools, commands, and IT concepts
- **Labs/** → Virtual machine setup and lab environment notes
